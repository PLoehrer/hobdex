# Hobdex
 
## Goals
 
### The app
A personal hobby tracking tool. Keep all your hobbies in one place, track the projects within each hobby, and see your progress at a glance. The home page shows a card for each hobby with its name, image, and stats. Clicking into a hobby will show its projects with details, status, notes, and more.
 
Planned features:
- Add, edit, and delete hobbies
- Add, edit, and delete projects within each hobby
- Upload custom images for hobbies
- Project status tracking (not started, in progress, completed)
- Notes and details per project
- User authentication (longer term)
 
### The learning
This project is being built as a hands-on way to learn modern frontend development. The developer comes from a C# / .NET / SQL background and is using this project to get comfortable with React, TypeScript, and the JavaScript ecosystem before eventually wiring up a familiar .NET backend.
